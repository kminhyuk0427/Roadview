# Changelog


## 1.2.5

* 모스키토 브로커를 통한 mqtt송수신 기능 구현
* 편의를 위하여 시스템 일시정지 기능을 추가
* mqtt 발행 중에 발생하던 크래시 현상을 해결
* 락을 잡아서 충돌 크래시를 해결
* count_manager.c에 불필요한 터미널 출력 코드 삭제
* 이전 포트 정보를 저장하여 재연결

## 1.1.3

* 객체 카운트 기능 추가
* 객체 카운트 후 출력에는 성공했으나 화면에 객체가 계속 존재하면 카운트가 계속 되는 문제를 해결
* 샘플 config파일을 가져온 후 모듈 엔진과 영상의 경로를 절대경로로 지정하여 실행 시간을 단축함
* 영상 출력 타일을 1로 지정하여 하나의 카메라만 표시되도록 수정
